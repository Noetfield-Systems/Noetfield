Code becomes cheap. Institutional software production does not.

AI may reduce the cost of creating code. It does not automatically solve authoritative requirements, hidden business rules, integration dependencies, security and access control, production reliability, rollback and recovery, long-term maintenance, accountability, or evidence of realised value.

The new enterprise risk is not only vendor lock-in. It is AI-generated shadow IT at institutional scale. The factory — not the individual application — is the durable infrastructure.

Decide first: retain · renegotiate · wrap · replace · retire

AI changes the build-versus-buy threshold. It does not eliminate disciplined build-versus-buy decisions.

DecisionMeaning
RetainVendor system remains economically and operationally justified.
RenegotiateKeep the system but reduce licences, scope or contract cost.
WrapPreserve the system of record; add a better AI-native workflow.
ReplaceRebuild a bounded application or operational module internally.
RetireRemove duplicated or low-value software.
